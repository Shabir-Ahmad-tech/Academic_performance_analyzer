# 🎓 Academic Performance Analyzer
### **Premium ICT Laboratory Final Project — UEAS Swat**

A high-end, minimalist student grade analysis tool designed for the **University of Engineering & Applied Sciences Swat**. This project automates the calculation of academic results across seven core subjects, providing instant performance insights with a premium UI/UX.

---

## 🚀 Overview
This application was developed as a final project for the **ICT Laboratory** (BS-AI, First Semester). It moves away from manual calculations, offering a digital-first approach to tracking student grades with precision and style.

## ✨ Key Features
- **7-Subject Tracking:** Specialized fields for ICT, Programming, Mathematics, English, Pak-Studies, Islamyat, and Fahm-e-Quran.
- **Instant Grading:** Real-time logic that calculates total marks, percentage, and assigned grades (A+ to F).
- **Premium Aesthetics:** Modern minimalist design using glassmorphism, high-end typography, and smooth transitions.
- **Responsive Architecture:** Fully optimized for desktops, tablets, and mobile devices.
- **Academic Integration:** Includes the full professional project report for institutional submission.

## 🛠️ Tech Stack
- **Frontend:** HTML5, CSS3 (Custom Design System)
- **Logic:** Vanilla JavaScript (ES6+)
- **Design Standard:** Premium Vibe Coding (Minimalist/Enterprise)
